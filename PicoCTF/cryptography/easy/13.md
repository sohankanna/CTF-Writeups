## Analysis and Solution 

The given cipher text is as follows

```text
cvpbPGS{abg_gbb_onq_bs_n_ceboyrz}
```

This is encrypted with ROT13. ROT13, which stands for "rotate by 13 places," is a specific type of Caesar cipher where the shift value is always 13

<img width="696" height="352" alt="ROT13" src="https://github.com/user-attachments/assets/16d90dcb-78a6-4b09-aff8-8663f45d57a7" />

We can use online tools itself to solve this, one such is https://www.dcode.fr/rot-13-cipher

Just input the cipher text and click decrypt to get the flag
